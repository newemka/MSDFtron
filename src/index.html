Font Size
Increase if too much empty space in created image; decrease if two images were created (Tooll3 supports only one atlas map per font). You might need to increase this value for fonts with lots of small details (i.e. a grunge font)
Texture Size
A higher value doesn't mean you'll get better result. Use high resolution for fonts with lots of small details (i.e. a grunge font) or if you need to pack a lot of glyphs (Chinese, Japanese, Korean...)
Distance Range
This will impact the softness of the font rendering and range of shadow. Too low causes aliasing. Too high artifacts.
Texture Padding
10 for 2048 is recommended
